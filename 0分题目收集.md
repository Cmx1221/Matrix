## 1. 25程设Ⅱ-周15-课堂1 — 比特串类型解析

- **类型**：实时编程题
- **状态**：已完成 / 已截止
- **分数**：0
- **链接**：https://matrix.sysu.edu.cn/course/private/2310286/assignment/2954214
- **限制**：1000 ms / 32 MB

### 题目描述

实现 `template<typename T> T decode_bits(const string& bits)` 模板函数，根据接收的比特字符串，还原为目标模板类型 T 的实际数值并返回。

- 32位类型（`int`/`float`）：接收 32 位二进制比特字符串
- 64位类型（`double`）：接收 64 位二进制比特字符串
- 浮点数遵循 IEEE 754 标准
- 比特串按**大端序**打包（`bits[0]` 是最高位）
- 输入多组数据：类型（INT/FLOAT/DOUBLE）+ 比特串
- 浮点数保留 4 位小数

### 样例

```
输入：
INT 00000000000000000000000000101010
FLOAT 00111111100000000000000000000000
DOUBLE 1011111111110000000000000000000000000000000000000000000000000000

输出：
42
1.0000
-1.0000
```

## 2. 25程设II-周14-课堂3 — 电子设备资产管理系统

- **类型**：实时编程题
- **状态**：已完成 / 已截止
- **分数**：0
- **链接**：https://matrix.sysu.edu.cn/course/private/2310286/assignment/2924019
- **限制**：1000 ms / 32 MB

### 题目描述

管理三种设备：Laptop（功耗60/寿命180）、Desktop（功耗120/寿命360）、Printer（功耗80/寿命240）。

要求实现多继承：`class Device`（抽象基类）+ `class Usable`（接口），派生 `Laptop final`、`Desktop final`、`Printer final`，统一保存在 `vector<Device*>` 中。

系统支持 11 种命令：
1. **PURCHASE type** — 购买设备
2. **TOTAL** — 输出总功耗
3. **COUNT_N type** — 某类型设备数量（必须用 `dynamic_cast`）
4. **COUNT_L type** — 某类型设备剩余寿命总和
5. **MAX_N** — 数量最多的设备类型（优先级 Laptop > Desktop > Printer）
6. **MIN_N** — 数量最少的设备类型
7. **MAX_L** — 总寿命最大的设备类型
8. **MIN_L** — 总寿命最小的设备类型
9. **USE type** — 找到 life 最大的该类型设备，执行 `life -= power`，若 life≤0 则删除
10. **TYPECHECK** — 统计各类型数量（必须用 `typeid`）
11. **END** — 结束

- 优先级固定：Laptop > Desktop > Printer
- 系统为空时多项输出 "NO"

## 3. 25程设II-周14-课堂2 — 动物园管理系统

- **类型**：实时编程题
- **状态**：已完成 / 已截止
- **分数**：0
- **链接**：https://matrix.sysu.edu.cn/course/private/2310286/assignment/2954095
- **限制**：1000 ms / 32 MB

### 题目描述

继承抽象类 `Animal`，实现四种动物：

| 动物 | 叫声 | 每天食物消耗 |
|------|------|-------------|
| Dog | Woof | 5 |
| Cat | Meow | 3 |
| Cow | Moo | 10 |
| Sheep | Baa | 7 |

支持 6 种命令：
- **ADD type** — 增加动物
- **SPEAK** — 所有动物按加入顺序发声
- **FEED** — 输出总食物消耗
- **COUNT type** — 统计某动物数量
- **TYPECHECK** — 用 RTTI（`dynamic_cast`/`typeid`）统计各动物数量，输出格式 `Dog:2 Cat:1 Cow:1 Sheep:0`
- **EXIT** — 结束，释放内存

第一行 `n` 表示操作数量，接下来 n 行命令。区分大小写。

## 4. 25程设Ⅱ-周7-课堂4 — 序列副本

- **类型**：实时编程题
- **状态**：已完成 / 已截止
- **分数**：0
- **链接**：https://matrix.sysu.edu.cn/course/private/2310286/assignment/2894773
- **限制**：1000 ms / 32 MB

### 题目描述

设计 `Seq` 类管理动态整数序列，包含：
- `int n` — 序列长度
- `int* data` — 动态数组指针
- `static int total` — 存在对象个数

需要实现：
- `Seq(int n=0)` — 构造函数，total+1
- `Seq(const Seq& other)` — 深拷贝构造函数，total+1
- `~Seq()` — 析构函数，释放内存，total-1
- `void input()` — 读入 n 个整数
- `void set(int index, int value)` — 修改元素
- `int longestNonDec() const` — 最长连续不下降子段长度
- `int diffCount(const Seq& other) const` — 两序列对应位置不同元素个数
- `static int getCount()` — 返回当前对象个数

输入：n → n个整数 → index value
输出：修改后原序列最长不下降长度 → 副本序列最长不下降长度 → 不同元素个数 对象数
