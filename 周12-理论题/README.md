# 程序设计Ⅱ - 周12 理论题

## 题目来源

程设Ⅱ - 周12 - 理论题

## 题目描述

第十二周理论题，多态、虚函数。

---

### 第1题 - 单选题

在C++中，多态意味着？

- **A.** 只具有单一形态的类
- **B.** 编译时决定行为的类
- **C.** 允许对象表现出多种行为形态的特性
- **D.** 仅具备静态行为的类

> **标准答案：C**

---
### 第2题 - 单选题

以下哪个概念允许你在C++中重用已编写的代码？

- **A.** 继承
- **B.** 多态
- **C.** 抽象
- **D.** 封装

> **标准答案：A**

---
### 第3题 - 单选题

在C++中，虚函数的主要目的是什么？

- **A.** 实现运行时多态
- **B.** 减少代码量
- **C.** 保证数据的安全性
- **D.** 提高代码的执行效率

> **标准答案：A**

---
### 第4题 - 单选题

虚函数是如何实现动态绑定（或称为迟绑定）的？

- **A.** 通过在编译时确定函数地址
- **B.** 仅在静态类型为基类时使用
- **C.** 通过在每个类中复制函数代码

### 第13题 - 单选题

以下代码中，哪个选项正确描述运行时行为？

```cpp
class Base { public: virtual void func() { std::cout << "Base"; } };
class Derived : public Base { public: void func() override { std::cout << "Derived"; } };
Derived d; Base b = d; b.func();
```

- **A.** 输出"Derived"
- **B.** 输出"Base"
- **C.** 编译错误
- **D.** 运行时崩溃

> **标准答案：B**
