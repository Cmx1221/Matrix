### 第1题 - 单选题

在 C++ 中，面向对象程序设计中的"多态"主要指的是：

- **A.** 一个类只能生成一种固定类型的对象
- **B.** 同一个操作或接口在不同对象上可以表现出不同的行为
- **C.** 一个函数只能被调用一次
- **D.** 所有类都必须拥有完全相同的数据成员

> **标准答案：B**

### 第2题 - 单选题

下列关于 C++ 多态的说法，正确的是：

- **A.** 函数重载属于运行时多态
- **B.** 虚函数机制主要用于实现运行时多态
- **C.** 多态只能通过模板实现
- **D.** 只要两个函数名称相同，就一定发生运行时多态

> **标准答案：B**

### 第3题 - 单选题

在 C++ 中，将基类成员函数声明为 virtual 的主要意义是：

- **A.** 使该函数只能在基类对象中调用
- **B.** 使该函数在编译时一定绑定到基类版本
- **C.** 允许通过基类指针或引用调用时，根据实际对象类型选择函数版本
- **D.** 使派生类不能重新定义该函数

> **标准答案：C**

### 第4题 - 单选题

下列哪种情况最能体现 C++ 的运行时多态？

- **A.** 同一个类中定义多个同名但参数不同的函数
- **B.** 通过基类指针调用虚函数，实际执行派生类中重写的版本
- **C.** 使用 int、double 等不同类型变量进行算术运算
- **D.** 在类中定义私有成员变量，限制外部访问

> **标准答案：B**

### 第8题 - 单选题

下列代码的输出结果是：
#include <iostream>
using namespace std;
class Base {
public:
virtual void show() {
cout << "Base";
}
};
class Derived : public Base {
public:
void show() {
cout << "Derived";
}
};
int main() {
Derived d;
Base* p = &d;
p->show();
return 0;
}

- **A.** Base
- **B.** Derived
- **C.** BaseDerived
- **D.** 编译错误

> **标准答案：B**
