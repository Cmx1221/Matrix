# 程序设计Ⅱ - 周14 理论题

## 题目来源

程设Ⅱ - 周14 - 理论题

## 题目描述

第十四周理论题，虚函数及其意义，多态的概念，多态与函数重载，多态与动态类型

---

### 第1题 - 单选题

关于多态，下列说法正确的是

- **A.** 一个对象同时占用多个内存空间
- **B.** 同一个接口表现出不同的行为
- **C.** 一个类只能有一个父类
- **D.** 一个函数只能被调用一次

> **标准答案：B**

---

### 第2题 - 单选题

下列关于纯虚函数的声明正确的是

- **A.** virtual void f();
- **B.** void f() = 0;
- **C.** virtual void f() = 0;
- **D.** void virtual f();

> **标准答案：C**

---

### 第3题 - 单选题

关于抽象类，错误的是（ ）

- **A.** 可以声明指针
- **B.** 可以声明引用
- **C.** 可以实例化对象
- **D.** 可以作为基类

> **标准答案：C**

---

### 第4题 - 不定项选择题

当Animal为抽象类时，下列代码中合法的是

- **A.** Animal a;
- **B.** Animal* p;
- **C.** Animal x = Dog();
- **D.** Animal* fun();

> **标准答案：B, D**

---

### 第5题 - 单选题

关于虚析构函数的说法正确的是

- **A.** 析构函数不能是虚函数
- **B.** 只有派生类需要虚析构
- **C.** 多态基类应该具有虚析构函数
- **D.** 虚析构会阻止对象析构

> **标准答案：C**

---

### 第6题 - 单选题

dynamic_cast 向下转换失败时，若目标类型是指针，则返回

- **A.** 0
- **B.** nullptr
- **C.** false
- **D.** 异常

> **标准答案：B**

---

### 第7题 - 单选题

关于纯虚函数，下列说法错误的是

- **A.** 使用 =0 声明
- **B.** 所在类一定是抽象类
- **C.** 纯虚函数必须实现函数体
- **D.** 子类可将其实现为普通函数

> **标准答案：C**

---

### 第8题 - 单选题

关于接口风格抽象类，下列说法错误的是

- **A.** 通常由纯虚函数组成
- **B.** 可以包含静态成员
- **C.** 可以作为多重继承的公共接口
- **D.** 可以包含普通数据成员

> **标准答案：D**

### 第9题 - 单选题

RTTI机制主要解决的问题是

- **A.** 编译时类型检查
- **B.** 运行时识别对象真实类型
- **C.** 动态分配内存

### 第10题 - 单选题

关于抽象类的描述，正确的是

- **A.** 抽象类可以直接创建对象
