### 第12题 - 单选题

以下代码中，哪一行会导致编译错误？
struct Base { virtual void foo(); };    // ①
struct A : Base { void foo() final; void bar() final; };  // ②③
struct B final : A { void foo() override; };  // ④
struct C : B {};  // ⑤

- **A.** ①
- **B.** 只有③
- **C.** ③ 和 ④ 和 ⑤
- **D.** ②

> **标准答案：C**
