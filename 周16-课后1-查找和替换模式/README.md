# 查找和替换模式

## 题目来源

程设Ⅱ - 周16 - 课后1（选做）

## 题目描述

你有一个单词列表words和一个模式pattern，你想知道words中的哪些单词与模式匹配。

如果存在字母的排列p ，使得将模式中的每个字母x替换为p(x) 之后，我们就得到了所需的单词，那么单词与模式是匹配的。（回想一下，字母的排列是从字母到字母的双射：每个字母映射到另一个字母，没有两个字母映射到同一个字母。）返回 words 中与给定模式匹配的单词列表，请按照输入顺序返回答案。若不存在匹配的单词列表则输出empty。

示例：

输入：words = ["abc","deq","mee","aqq","dkd","ccc"], pattern = "abb"
输出：["mee","aqq"]

解释：
"mee"与模式匹配，因为存在排列 {a -> m, b -> e, ...}。
"ccc"与模式不匹配，因为 {a -> c, b -> c, ...} 不是排列，a和b映射到同一个字母。

注意：pattern.length = words[i].length

## 输入数据

- 第一行输入单词列表，多个单词之间用空格分隔
- 第二行输入模式

## 输出数据

与给定模式匹配的单词列表，多个单词之间用空格分隔，若不存在匹配的单词列表则输出empty

## 样例

### 输入示例1

```
abc deq mee aqq dkd ccc
abb
```

### 输出示例1

```
mee aqq
```

### 输入示例2

```
a b c
a
```

### 输出示例2

```
a b c
```
