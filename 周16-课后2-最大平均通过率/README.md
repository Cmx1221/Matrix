# 最大平均通过率

## 题目来源

程设Ⅱ - 周16 - 课后2（选做）

## 题目描述

一所学校里有 `n` 个班级，每个班级里有一些学生，现在每个班都会进行期末考试。

给你一个二维数组 `classes` ，其中 `classes[i] = [pass_{i}, total_{i}]` ，表示你提前知道了第 `i` 个班级总共有 `total_{i}` 个学生，其中有 `pass_{i}` 个学生会通过考试。

给你一个整数 `extraStudents` ，表示额外有 `extraStudents` 个学生，他们一定能通过期末考。你需要给这 `extraStudents` 个学生每人都安排一个班级，使得所有班级的平均通过率最大。

一个班级的通过率等于这个班级通过考试的学生人数除以这个班级的总人数。平均通过率是所有班级的通过率之和除以班级数目。

输出是在安排这 `extraStudents` 个学生去对应班级后的最大平均通过率。

## 输入输出

输入：
- 第1行为 `extraStudents` ，表示额外有 `extraStudents` 个学生，他们一定能通过期末考
- 第2行为 `n` ，表示有 `n` 个班级
- 第3行到第n+2行每行第一个数表示第i个班级只有 `pass_{i}` 个学生可以通过考试，第二个数表示第i个班级总共有 `total_{i}` 个学生

输出：
- 最大平均通过率res

## 样例

### 示例 1

输入：
```
2
3
1 2
3 5
2 2
```

输出：
```
0.783
```

解释：你可以将额外的两个学生都安排到第一个班级，平均通过率为 (3/4 + 3/5 + 2/2) / 3 = 0.783

### 示例 2

输入：
```
4
4
2 4
3 9
4 5
2 10
```

输出：
```
0.535
```

## 部分数据范围

- `1 <= classes.length <= 10^{5}`
- `1 <= pass_{i} <= total_{i} <= 10^{5}`
- `1 <= extraStudents <= 10^{5}`
