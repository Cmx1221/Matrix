# 只出现一次的数字

## 题目来源

程设Ⅱ - 周16 - 课堂1

## 题目描述

给你一个非空整数数组 `nums` ，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

## 输入数据

- 数组长度
- 数组元素，用空格隔开

## 输出数据

只出现一次的数字

## 样例

### 输入示例1

```
3
2 2 1
```

### 输出示例1

```
1
```

### 输入示例2

```
5
4 1 2 1 2
```

### 输出示例2

```
4
```

### 输入示例3

```
1
1
```

### 输出示例3

```
1
```

## 限制

- 时间限制：500 ms
- 内存限制：32 MB
