### 第1题 - 单选题

在C++中，string是属于以下哪种类型？

- **A.** 基本数据类型（如 int, float）
- **B.** C 风格字符串类型
- **C.** C++ 标准库类类型（如 std::string）
- **D.** 宏定义类型

> **标准答案：C**

---
### 第2题 - 单选题

以下哪个是正确的 string对象初始化方式？

- **A.** string s = 'hello';
- **B.** string s = "hello";
- **C.** string s = hello;
- **D.** D. string s('hello');

> **标准答案：B**

- **B.** s1 默认为空字符串，s2 初始化为 "C++"
