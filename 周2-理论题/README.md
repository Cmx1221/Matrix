### 第1题 - 单选题

在C++中，string是属于以下哪种类型？

- **A.** 基本数据类型（如 int, float）
- **B.** C 风格字符串类型
- **C.** C++ 标准库类类型（如 std::string）
- **D.** 宏定义类型

> **标准答案：C**

---

### 第2题 - 单选题

以下哪个是正确的 string对象初始化方式？

- **A.** string s = 'hello';
- **B.** string s = "hello";
- **C.** string s = hello;
- **D.** D. string s('hello');

> **标准答案：B**

---

### 第3题 - 单选题

下面关于 string对象 s1和 s2的说法，正确的是：
string s1;
string s2 = "C++";

- **A.** s1 是未初始化的，s2 是空字符串
- **B.** s1 默认为空字符串，s2 初始化为 "C++"
- **C.** s1 和 s2 都是未初始化的
- **D.** s2 是 C 风格字符串，不是 string 对象

> **标准答案：B**

---

### 第4题 - 单选题

C++ 支持函数重载，指的是：

- **A.** 多个函数可以有相同名字，但参数列表不同
- **B.** 一个函数可以有多个返回值
- **C.** 函数可以在多个文件中定义
- **D.** 函数必须使用相同的参数类型

> **标准答案：A**

---

### 第5题 - 不定项选择题

以下哪个是 string::find函数的正确用法示例？

- **A.** s.find("ab", 2, 3)
- **B.** s.find("ab", 2)
- **C.** s.find(123)
- **D.** s.find()

> **标准答案：B, A, C**

### 第6题 - 单选题

在 C++11 中，constexpr主要用于：

- **A.** 表示变量是只读的
- **B.** 用于定义宏
- **C.** 表示函数或变量能在编译时求值（常量表达式）
- **D.** 用于替代 #define

> **标准答案：C**

### 第7题 - 单选题

关于 auto关键字，以下说法正确的是？

- **A.** 在 C++11 中auto 可以用于函数参数类型
- **B.** auto 可以表示任意类型，无需推导
- **C.** auto 定义的变量不需要初始化
- **D.** auto 让编译器根据初始化表达式自动推导变量类型

> **标准答案：D**

### 第8题 - 单选题

以下关于 C++ 中 string 类的说法，错误的是？

- **A.** string 类对象可以直接用 +运算符进行字符串拼接
- **B.** string::length() 返回类型为 int
- **C.** string 类提供了 find()方法用于查找子串
- **D.** 可以使用 c_str()方法获取与 C 风格字符串兼容的字符数组

> **标准答案：B**

### 第9题 - 单选题

vector<int> vec = {1, 2, 3};
auto it = vec.begin();

- **A.** 让 it 的类型成为 vector
- **B.** 显式声明 it 为 int 类型
- **C.** 让编译器根据 vec.begin() 的返回值自动推导 it 的类型
- **D.** 让 it 指向 vec 的最后一个元素

> **标准答案：C**

### 第10题 - 单选题

以下关于 C++ 中的 string::substr()方法，说法正确的是？

- **A.** substr()用于删除字符串中的某一部分
- **B.** substr(pos, len)返回从位置 pos 开始，长度为 len 的子字符串
- **C.** substr()只能返回整个字符串
- **D.** substr()方法会修改原字符串

> **标准答案：B**
