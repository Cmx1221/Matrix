### 第1题 - 单选题

关于 C/C++ 程序内存布局，下列说法正确的是

- **A.** 自动变量通常存放在堆区
- **B.** 动态变量通常存放在栈区
- **C.** 自动变量通常存放在栈区
- **D.** 字符串字面量通常可以随意修改

> **标准答案：C**

- **B.** 输出 s2
- **C.** 修改 s2[0] = 'X'
- **D.** 修改 s1[0] = 'X'

### 第4题 - 单选题

### 第5题 - 单选题

- **A.** delete 后指针会自动变成 nullptr
